# Contributing

Contributions are welcome for educational improvements, additional synthetic use cases, clearer explanations, or better visualization layers.

## Suggested contribution ideas

- add rendered notebook outputs
- add comparison notebooks for ComBat or RUV-style approaches
- add tests for helper functions
- add HTML exports
- improve figures and explanatory diagrams
- expand the methylation workflow
- add single-cell or proteomics examples

## Style goals

Please keep contributions:
- readable
- well commented
- modular
- teaching oriented
